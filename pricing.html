How quotes are set

Pricing usually depends on complexity, not just file count.

Two projects with the same number of samples can require very different effort depending on design, metadata quality, and interpretation needs.

Design complexity

More groups, paired designs, batches, or covariates usually require more planning and validation.

Analysis depth

A quick result review is very different from a manuscript-ready package with multiple custom figures.

Turnaround speed

Expedited requests can be accommodated when scheduling allows, but they should be scoped separately.
